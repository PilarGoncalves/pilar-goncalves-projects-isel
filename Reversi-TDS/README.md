# Reversi (Othello) – Desktop Application

This project consists of a desktop implementation of the **Reversi** board game, developed as part of the course **Software Development Techniques**.

The main goal of the project was to apply good software development practices, including modular design, clear separation of concerns, and collaborative development.

---

## Project Overview

The game was implemented in **Kotlin** and supports two different modes of interaction:

- **Terminal-based version** – allows playing Reversi directly in the command line
- **Desktop graphical version** – built using **Composable UI**, enabling interactive gameplay on a personal computer

Additionally, the project includes support for **remote multiplayer**, allowing two players to play from different computers using a shared database.

---

## Technologies & Tools

- **Programming Language:** Kotlin  
- **User Interface:** Composable (desktop UI)  
- **Database:** MongoDB  
- **Development Environment:** IntelliJ IDEA  

---

## Multiplayer Functionality

To support games between players on different machines, **MongoDB** was used as a backend database.  
The database is responsible for storing and synchronizing the game state, enabling turn-based gameplay across multiple computers.
